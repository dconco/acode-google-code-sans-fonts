# Google Sans Code for Acode

This extension brings the clean, professional, and highly legible Google Sans Code typeface to your Acode editor. Optimized for programming, it provides excellent character distinction and a modern aesthetic for your mobile development environment.

### Features

**Modern Design:** Enjoy the sleek look of Google's custom monospace font.

**Readability:** High legibility for long coding sessions.

**Easy Integration:** One-click activation within Acode.

### Usage

After installation, go to Settings > Editor Settings > Font Family and select "Google Sans Code".
If it doesn't appear immediately, a quick restart of the app will refresh the list.
